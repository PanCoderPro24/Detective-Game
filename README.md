Detective Game

Description:
A text-based Python detective game where the player must solve a murder by questioning suspects, gathering clues, and making a logical accusation.

Current Version (v0.3 – Alpha):

Statement-based dialogue system for each suspect
Investigation system with limited attempts
Clue system tied directly to suspects
Randomized killer and clue order for replayability
Deduction-based gameplay with no hints or guidance

Gameplay:

Talk to suspects to hear their statements
Investigate to uncover clues
Compare statements with evidence
Identify contradictions and determine the killer

Features:

No hand-holding: players must deduce the answer themselves
Clean command-line interface
Replayable with different outcomes each run

How to Play:

Run the Python script in a terminal
Choose actions from the menu
Use logic and evidence to identify the killer
Make an accusation before investigations run out

Detective Game

Description:
A text-based Python game where the player is a detective solving a murder case. The player interrogates suspects, collects clues, and makes an accusation.

Current Version (v0.1):

Basic talk system implemented.
Accusation system with win/lose outcome.
Menu interface: Talk, Investigate (placeholder), View clues, Accuse.
Input validation for menu and suspect names.

How to Play:

Run the Python script in a terminal.
Use the menu to interact with suspects or make an accusation.
Deduce the killer logically to win.
